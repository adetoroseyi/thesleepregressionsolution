# The Sleep Regression Solution

A high-converting sales page for digital ebook products.

## Tech Stack

- **Frontend**: Next.js 14 + Tailwind CSS
- **Hosting**: Vercel
- **Payments**: Stripe (coming soon)
- **Database**: Supabase (coming soon)
- **Email**: Resend (coming soon)

## Quick Start

### Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Deploy to Vercel

This project is configured to deploy automatically via Vercel when you push to GitHub.

## Project Structure

```
├── app/
│   ├── globals.css      # Tailwind + custom styles
│   ├── layout.js        # Root layout with metadata

├── public/

├── package.json
```
